Testing the Womier K66

If everything worked you should notice some changes, test them now!

  1. The caps lock key should now be a FN key.
  2. You can use the keybind CAPS + S + 1/2 to change the RGB effect.
  3. You can use the keybind CAPS + S + Insert/Delete to change the brightness of the RGB

If you dislike the default keybinds or layout, you will need to modify the QMK keymap file located at keyboards/womier/k66/keymaps/default/keymap.c. Thankfully to rebuild you just need to repeat the make command when we built QMK. Push the button to go to QMK's website to learn more about keymaps. If you ever have any questions please join our discord server on the top of the page. Thank you so much for reading.
