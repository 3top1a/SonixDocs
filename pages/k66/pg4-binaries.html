Installing the pre-compiled Womier Flasher

Womier Flasher is a simple tool created by xyz which assists with flashing the pre-compiled firmware onto the board. We have built a version of it so you don't have to install all the dependencies and such.

1.) Download Womier Flasher binary

Press the big button bellow and download the required file for your system.

Simply run the executable and go onto the next step.
